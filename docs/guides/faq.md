# FAQ

## Is RDS an Obsidian vault?

No. RDS is a local-file research discussion system. Obsidian can open and visualize an RDS Project, but it is not required.

## Is RDS a Claude plugin?

No. RDS has a Claude Cowork adapter, but the core file protocol is provider-neutral.

## Can multiple AI tools work on the same RDS Project?

Yes, if each tool follows `99_meta/rds_agent_contract.md` and records what it read, changed, and inferred.

## Should every AI chat become a discussion log?

No. Only bounded, useful research discussions should become logs. Small clarifications can remain unlogged unless they produce durable context.
