# Obsidian Integration

[English](obsidian_integration.md)

Obsidian은 RDS Project를 탐색하고 시각화하기 위한 선택적 인터페이스입니다.

## 유용한 기능

- backlinks
- graph view
- wiki links
- Dataview-compatible YAML fields
- `_index.md` 파일을 Map of Content처럼 사용

## 원칙

Obsidian-enhanced mode는 일반 Markdown 파일로도 유지될 수 있는 정보만 담아야 합니다. RDS Project 폴더가 source of truth이며, Obsidian은 그 위에 얹는 탐색 도구입니다.
