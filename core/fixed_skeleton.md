# Fixed Skeleton

Every RDS Project must contain these folders:

```text
00_context/
04_discussions/
05_literature/
99_meta/
```

## 00_context

Durable project memory: user profile, project identity, decisions, assumptions, evidence, open questions, and glossary.

## 04_discussions

Structured logs of research discussions. Each log should include topic, source files, observations, hypotheses, decisions, next actions, and unresolved questions.

## 05_literature

Papers, citation notes, reading summaries, and literature-grounded claims.

## 99_meta

RDS runtime configuration, scaffold decisions, agent contract, safety policy, and template version.
