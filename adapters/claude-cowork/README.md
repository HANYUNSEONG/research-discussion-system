# Claude Cowork Adapter

Claude Cowork is the recommended first user experience for non-developer researchers because it combines conversational discussion, local file access, project memory, and plugin skills.

Use the repository root as the Claude plugin package. The plugin manifest is `.claude-plugin/plugin.json`, and workflow skills live in `skills/`.

## Recommended Role

- main research discussion surface
- setup interview
- discussion logging
- context resume
- literature/data-grounded brainstorming

## Boundary

Cowork is not the source of truth. RDS Project files are the source of truth.
