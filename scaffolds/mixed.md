# Mixed Scaffold

Use the mixed scaffold when the project has multiple distinct work modes, such as wet lab plus computational analysis.

## Default Folders

```text
01_research_materials/
02_data/
03_analysis/
```

Use these default folders first. Add mode-specific folders after setup only when the project needs a clearer handoff between work modes.

## Discussion Cues

These are not setup questions. Use them only when relevant during a discussion.

- Which work modes are included?
- Which mode is primary right now?
- Where do handoffs happen between modes?
- What outputs from one mode become inputs to another?
- Which vocabulary or file types need special handling?
